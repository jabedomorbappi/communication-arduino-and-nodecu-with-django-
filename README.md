### Features

- **Dual sensor paths: **Arduino + NodeMCU** reading same IR sensors simultaneously  
-Latency measurement between Arduino → Serial → NodeMCU → Web vs Direct NodeMCU → Web  
-Three relays with visual feedback (Arduino, NodeMCU, Piezo-powered)  
-Real-time dashboard with **8 advanced Plotly charts**  
-Common + Individual relay control  
-Connection status detection (Active / Disconnected)  
-Professional responsive design with **GSAP scroll animations**  
-Time range analytics (5min to 7 days)  
-Team page with smooth entrance effects
